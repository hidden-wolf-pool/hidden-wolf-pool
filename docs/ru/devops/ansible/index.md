**Ansible** — это инструмент автоматизации с открытым исходным кодом, созданный для упрощения управления конфигурациями, развёртывания приложений, оркестрации инфраструктуры и автоматизации задач на множестве систем.

- **Язык:** 🐍Python, 🔫YAML
- [Документация](https://docs.ansible.com/ansible/latest/index.html)
- [GitHub](https://github.com/ansible/ansible)
